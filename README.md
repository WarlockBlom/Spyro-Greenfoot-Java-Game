# Spyro-Greenfoot-Java-Game
A 2D Spyro-inspired game developed in Java using Greenfoot, built as an introduction to object-oriented programming and interactive game logic. The project features player movement, enemy interactions, gem collection, collision detection, and score tracking within a structured class-based architecture.
